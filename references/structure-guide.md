# Blog Article Structure Guide

Complete specifications for each article element.

---

## 1. Category Label

Small wayfinding element above the title.

**Specs:**
- Font: Cormorant, uppercase
- Size: 12px
- Letter spacing: 0.15em
- Position: Centered
- Optional: Decorative horizontal lines (40px wide, 1px)

---

## 2. Title (H1)

Dominant page element.

**Specs:**
- Font: Cormorant Garamond
- Size: 48–60px (responsive)
- Weight: 500
- Line height: 1.15
- Alignment: Center
- Bottom margin: 24px

---

## 3. Subtitle / Deck

Expands on headline promise.

**Specs:**
- Font: Cormorant
- Size: 20px
- Weight: 300
- Line height: 1.6
- Color: Muted
- Max width: 540px
- Alignment: Center
- Bottom margin: 32px

---

## 4. Author Block

Byline and metadata.

**Components:**
- Avatar: 40–44px circle
- Author name: 15–16px, medium weight
- Publication date: 13–14px, muted
- Reading time: 13–14px, muted, with clock icon

**Layout:** Centered, horizontal, flexible gaps.

---

## 5. Featured Image

Hero visual setting emotional tone.

**Specs:**
- Aspect ratio: 16:9
- Width: Full container (can extend beyond text column)
- Border radius: 8px
- Bottom margin: 56px

**Caption:**
- Font: Cormorant, italic
- Size: 14px
- Color: Muted
- Alignment: Center
- Top margin: 16px

---

## 6. Lead Paragraph

First paragraph, styled larger.

**Specs:**
- Font: Cormorant
- Size: 24px
- Weight: 300
- Line height: 1.7
- Bottom margin: 40px

---

## 7. Body Text

Standard paragraphs.

**Specs:**
- Font: Cormorant
- Size: 20px
- Weight: 400
- Line height: 1.7
- Max width: 650px
- Paragraph spacing: 28px
- Optimal line length: 65–75 characters

---

## 8. Section Heading (H2)

Major thematic breaks.

**Specs:**
- Font: Cormorant Garamond
- Size: 32px
- Weight: 500
- Line height: 1.3
- Top margin: 64px
- Bottom margin: 24px

---

## 9. Subheading (H3)

Second-level hierarchy.

**Specs:**
- Font: Cormorant Garamond
- Size: 24px
- Weight: 500
- Line height: 1.4
- Top margin: 48px
- Bottom margin: 20px

---

## 10. Blockquote

Direct quotations from external sources.

**Specs:**
- Background: Subtle tint (5–10% opacity)
- Left border: 3–4px solid accent
- Border radius: 0 left, 8px right
- Padding: 24px vertical, 32px horizontal
- Vertical margin: 40px

**Quote text:** Cormorant, italic, 20px
**Citation:** Cormorant, regular, 14–16px, muted, format: — Author Name

---

## 11. Pull Quote

Highlighted key phrase.

**Specs:**
- Alignment: Center
- Padding: 40px vertical
- Vertical margin: 56px
- No background

**Quote text:**
- Font: Cormorant Garamond, italic
- Size: 28px
- Line height: 1.5

**Decoration:**
- Large quotation mark above/behind
- Size: 80–120px
- Opacity: 10–15%

**Attribution:** Cormorant, regular, 14px, muted, 0.05em tracking

---

## 12. Callout Box

Supplementary information.

**Specs:**
- Background: Subtle gradient or tint (5–10% opacity)
- Border: 1px solid at 20% opacity (optional)
- Border radius: 12px
- Padding: 32px
- Vertical margin: 48px

**Title:** Cormorant Garamond, 20px, semibold
**Body:** Cormorant, 18px

---

## 13. Content Image

In-article visuals.

**Specs:**
- Aspect ratio: Flexible (3:2 recommended)
- Width: Full content or breakout
- Border radius: 8px
- Vertical margin: 48px

**Caption:** Cormorant, italic, 14px, muted, centered

---

## 14. Bullet List

List content (use sparingly).

**Specs:**
- Left padding: 24px
- Item spacing: 12px
- Vertical margin: 32px

**Bullet:** Circle, 8px, accent color
**Item text:** Cormorant, 20px, 1.7 line height

---

## 15. Section Divider

Visual break between major sections.

**Specs:**
- Vertical margin: 64px
- Alignment: Center

**Components:**
- Horizontal lines: 1px height, 60px wide
- Symbol: ✦ or similar, accent color
- Gap: 16px

---

## Content Width

- Body text column: 650px max
- Breakout elements: 800–900px

---

## Visual Rhythm Pattern

Alternate element types:

Text → Image → Text → Quote → Text → Divider → Text → Callout → Text

---

## Mobile Adjustments

- Body text: Minimum 16px
- Reduce spacing proportionally
- Featured/content images: Full width, no border radius
- Pull quotes: Reduce to 22–24px
- Side padding: 20px minimum
