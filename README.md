# Blog Article Layout Skill

A Claude skill for creating beautifully structured blog articles with proper typography hierarchy and visual rhythm.

## Typography

- **Headings:** Cormorant Garamond
- **Body:** Cormorant

## Elements

1. Category Label
2. Title (H1)
3. Subtitle
4. Author Block
5. Featured Image
6. Lead Paragraph
7. Body Text
8. Section Heading (H2)
9. Subheading (H3)
10. Blockquote
11. Pull Quote
12. Callout Box
13. Content Image
14. Bullet List
15. Section Divider

## Installation

### Claude Code

```bash
mkdir -p .claude/skills
git clone https://github.com/YOUR_USERNAME/blog-article-layout-skill.git .claude/skills/blog-article-layout
```

### Global (all projects)

```bash
mkdir -p ~/.claude/skills
git clone https://github.com/YOUR_USERNAME/blog-article-layout-skill.git ~/.claude/skills/blog-article-layout
```

## Usage

Ask Claude to format content as a blog article:

- "Format this as a blog article"
- "Apply blog layout to this content"
- "Create a blog post from this draft"
